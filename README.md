# 🦈 Wi-Fi Packet Sniffer (Python + Scapy + Wireshark)

This project is a simple **Wi-Fi Packet Sniffer** built using **Python** and **Scapy**.  
It captures live network traffic, displays source and destination IP addresses, and can be extended to show deeper packet details.  

---

## 🚀 Features  
✅ Captures live Wi-Fi traffic  
✅ Displays source and destination IP addresses  
✅ Filters HTTP/HTTPS traffic (Optional)  
✅ Works with Scapy + Wireshark  

---

## 🛠️ Requirements  
- Python 3.8 or above  
- Scapy  
- Npcap (for Windows)  
- Wireshark (Optional)
